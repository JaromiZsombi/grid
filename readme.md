# Git verziókezelés

## Helyi repo létrehozása

- helyi repo inicializálása
    > git init
- ellenőrzés:
    > git status
- előkészítjük az commit-ra (a verzió létrehozására):
    > git add .
- ellenőrzés:
    >git status
- commit:
    >git commint -m "first commit"
- a commit-ok listázása
    > git log
